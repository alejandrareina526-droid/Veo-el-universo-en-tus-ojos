# 🌌 Mi universo tiene tu nombre ❤️

Una página web creada con mucho cariño para guardar recuerdos, momentos especiales y pequeños detalles de una historia de amor.

Este proyecto representa un universo digital lleno de emociones, donde cada sección guarda un pedacito de una historia: una carta escrita con el corazón, mensajes secretos, una constelación de recuerdos, fotografías y momentos únicos.

✨ Características:

* 🌙 Diseño inspirado en el universo y las estrellas.
* 💌 Carta animada con efecto de escritura.
* 🔐 Mensajes secretos interactivos.
* ⭐ Constelación de recuerdos con mensajes personalizados.
* 📸 Galería de momentos especiales.
* ❤️ Animaciones y detalles románticos.

🛠️ Tecnologías utilizadas:

* HTML5
* CSS3
* JavaScript

Creado con dedicación, creatividad y mucho amor. 💙🌌
